
# Firmware for SplittedSpace Corne keyboards

## Corne

**corne_vial.hex**

*Features:*

* Macros
* OLED
* Lighting:
  * BREATHING
  * ALTERNATING
  * RAINBOW_MOOD
  * RAINBOW_SWIRL
  * SNAKE

*Features:*

* Macros
* Combos
* Tap Dance
* Lighting:
  * BREATHING
  * ALTERNATING
  * RAINBOW_MOOD
  * RAINBOW_SWIRL
  * SNAKE
